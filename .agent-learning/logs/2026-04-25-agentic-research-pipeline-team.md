# Agentic Research Pipeline Team Creation

## Observation

The user requested a specialized team of agents that can organize and operate an agentic research pipeline for collecting real validation statistics for PsyCalc.

## Action

Created dedicated agent specifications for:

- `research-orchestrator`
- `experiment-designer`
- `data-pipeline-engineer`
- `ethics-and-consent-reviewer`
- `literature-researcher`

Existing agents remain part of the pipeline:

- `psychometrics-methodologist`
- `statistical-validation-agent`
- `source-provenance-auditor`
- `empirical-claims-caveats-reviewer`
- `scoring-calibration-researcher`
- `socionics-researcher`
- `psychosophy-researcher`
- `temporistics-researcher`
- `agent-improvement-steward`

## Safeguards

- New agents use review-gated workflows.
- Human approval is required for participant-facing studies, sensitive data, publication, scoring changes, and agent instruction changes.
- The team is designed to test PsyCalc hypotheses rather than convert them into deterministic claims.
