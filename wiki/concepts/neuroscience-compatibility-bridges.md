# Neuroscience Compatibility Bridges

## Status

This page maps the project's **Strategic / Operational / Tactical** compatibility heuristic to adjacent constructs in neuroscience and cognitive science.

These are **process-level analogies**, not validation of Socionics, Psychosophy, or Temporistics.

Do not use this page for direct brain-type mapping.

Unsafe claims:

- “Temporistics is the default mode network.”
- “Psychosophy Will is prefrontal cortex.”
- “Socionics type equals a brain network.”
- “1E is amygdala activity.”

Safe claims:

- “This project level resembles a class of cognitive-affective processes.”
- “This neuroscience construct can inspire validation tasks or scenario design.”
- “This evidence is an external mechanism bridge, not a type marker.”

## Core Mapping

| Project level | Typology layer | Neuroscience bridge | What it helps model |
|---------------|----------------|---------------------|---------------------|
| **Strategic** | Temporistics | default mode network, autobiographical memory, episodic future thinking, narrative identity | life trajectory, meaning, past-future continuity, temporal self |
| **Operational** | Psychosophy | executive control, salience network, emotion regulation, reward/valuation, stress/fatigue | action organization, priorities, pressure, effort, self-regulation |
| **Tactical** | Socionics | social cognition, mentalizing, conversation, turn-taking, interpersonal synchrony | situated understanding, information exchange, repair, interaction rhythm |

Compact formula:

```text
Strategic neuroscience bridge
= temporal self + autobiographical memory + prospection + meaning-making

Operational neuroscience bridge
= executive control + salience/valuation + regulation + effort allocation

Tactical neuroscience bridge
= social cognition + prediction + conversation repair + interpersonal synchrony
```

## Strategic Bridges

Strategic compatibility asks whether two people can share a long-horizon meaning and temporal frame.

### Default Mode Network / Self-Projection

The default mode network (DMN) is often discussed in relation to self-referential processing, autobiographical memory, future simulation, and narrative self.

Useful concepts:

- self-projection
- autobiographical memory
- episodic future thinking
- narrative identity
- mental time travel
- constructive episodic simulation

Authors / search anchors:

- Marcus Raichle
- Randy Buckner
- Jessica Andrews-Hanna
- Daniel Schacter
- Donna Rose Addis
- Endel Tulving
- Demis Hassabis
- Eleanor Maguire
- Karl Szpunar
- Arnaud D'Argembeau
- Thomas Suddendorf
- Michael Corballis
- `default mode network autobiographical memory future thinking`
- `constructive episodic simulation hypothesis`
- `mental time travel Tulving Suddendorf Corballis`

Project use:

- bridge to Temporistics Past/Future/Eternity
- scenario design for life direction, memory, future planning, shared narrative
- possible validation tasks around autobiographical memory and future simulation

Limit:

- DMN is not “Temporistics in the brain”. It is a broad network involved in multiple self/social/temporal tasks.

## Operational Bridges

Operational compatibility asks whether two people can organize action, pressure, roles, effort, and regulation together.

### Executive Control / Action Selection

Executive control research studies working memory, inhibition, task switching, goal maintenance, conflict monitoring, and action selection.

Useful concepts:

- frontoparietal control network
- cognitive control
- conflict monitoring
- action selection
- effort allocation
- agency and volition

Authors / search anchors:

- Earl Miller
- Jonathan Cohen
- Michael Posner
- John Duncan
- Matthew Botvinick
- `executive control network frontoparietal control`

Project use:

- bridge to Psychosophy Logic/Will as operational domains
- scenario design for decisions, pressure, planning, and task coordination

## Tactical Bridges

Tactical compatibility asks whether two people can read, exchange, repair, and synchronize situation information.

### Social Cognition / Mentalizing

Social neuroscience studies theory of mind, mentalizing, empathy, social inference, and how people model others' intentions and states.

Useful concepts:

- theory of mind
- mentalizing network
- empathy
- social prediction
- pragmatic inference
- joint attention

Authors / search anchors:

## Strong Analogies

| Bridge | Why it is useful |
|--------|------------------|
| Temporistics ↔ autobiographical memory / future thinking | both involve past-future self-continuity and mental time travel |
| Temporistics ↔ DMN | useful for self-projection and narrative self, with caveats |
| Psychosophy ↔ executive control / action organization | operational level is directly about action, control, and priorities |
| Psychosophy ↔ salience / valuation | both help model what becomes important enough to act on |
| Socionics ↔ social cognition / mentalizing | useful for situated understanding and partner-modeling |
| Socionics ↔ conversation / turn-taking | strong bridge for interaction simulations and observable repair patterns |

## Weak Analogies And Caveats

| Caveat | Why it matters |
|--------|----------------|
| No direct brain-type mapping | there is no validated neural phenotype for Socionics, Psychosophy, or Temporistics types |
| Avoid region reductionism | complex behavior is networked, dynamic, task-dependent, and context-sensitive |
| fMRI correlations are not personality causes | brain activation does not prove typological structure |
| Neuroscience vocabulary must not launder weak typology claims | “neural” language can make speculative claims sound stronger than they are |
| Networks are cross-level | DMN, salience, and control networks can participate in multiple project levels |
| Compatibility score should not be derived from neuroscience directly | neuroscience is a secondary explanatory/validation layer |

## Scenario Design Implications

Neuroscience can help generate realistic tasks and stressors for simulations.

| Scenario | Neuroscience bridge | Project level |
|----------|---------------------|---------------|
| Partners imagine a shared future | episodic future thinking / prospection | Strategic |
| Conflict about past mistakes | autobiographical memory / narrative identity | Strategic |
| One partner freezes under pressure | stress effect on executive function | Operational |
| Emotional escalation during decision-making | emotion regulation / salience | Operational |
| Fatigue makes small chores explosive | allostatic load / interoception / effort | Operational |
| Misread text message causes conflict | pragmatic inference / social prediction | Tactical |
| Conversation repeatedly fails to repair | turn-taking / repair / interactive alignment | Tactical |
| Interaction feels energizing or draining | interpersonal synchrony / arousal coupling | Tactical + Operational |
