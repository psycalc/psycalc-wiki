# Psychosophy Model

## Overview

The Psychosophy model describes personality through four aspects and their relative strengths.

## The Four Aspects

| Aspect | Russian | Description |
|--------|--------|-------------|
| Physics | Физика | Practical, concrete, sensory |
| Logic | Логика | Analytical, systematic |
| Will | Воля | Determination, willpower |
| Emotion | Эмоция | Emotional expression |

## Type Structure

Each type is represented by a 4-digit code:
- 1st digit: Physics strength (1-4)
- 2nd digit: Logic strength (1-4)
- 3rd digit: Will strength (1-4)
- 4th digit: Emotion strength (1-4)
