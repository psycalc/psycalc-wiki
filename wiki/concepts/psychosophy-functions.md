# Psychosophy Function Blocks

## Overview

Each Psychosophy type has four aspects with different expression levels.

## Function Expression Levels

| Level | Name | Description |
|-------|------|-------------|
| 1st | Excessive | Main support, dominant, result-oriented |
| 2nd | Normative | Balanced, supporting |
| 3rd | Inadequate | Challenging, underdeveloped |
| 4th | Insignificant | Minimal impact |
