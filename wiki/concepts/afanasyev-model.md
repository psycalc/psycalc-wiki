# Afanasyev's Psychosophy Model

## Overview

Alexander Afanasyev's model, also known as **Psychosophy** or **Psychoyoga**, is a theory of personality that describes the relationship between an individual and the categories of time: Present, Past, Future, and Eternity.

## The Four Aspects

The model includes four functions that represent direct ways for a person to influence the world:

| Aspect | Russian | Description |
|--------|--------|-------------|
| Physics | Физика (Φ) | Practical, concrete, sensory information |
| Logic | Логика (Л) | Analytical, systematic reasoning |
| Will | Воля (В) | Determination, willpower, decisiveness |
| Emotion | Эмоция (Э) | Emotional expression, empathy |

## Function Expression Levels

In Afanasyev's model, each function has a specific expression level:

| Position | Expression | Characteristics |
|----------|------------|-----------------|
| 1st | Excessive | Main support, static, immovable, result-oriented, confident but cautious |
| 2nd | Normative | Balanced, appropriate, supporting |
| 3rd | Inadequate | Challenging, underdeveloped, may struggle with nuances |
| 4th | Insignificant | Minimal impact, often overlooked |

## First Function Description

The first function in Psychosophy is characterized as:

- **Support:** Main support for each person, something static and immovable
- **Confidence:** Origin of "confidence in tomorrow" — a person always relies on it
- **Result-oriented:** Focused on outcomes and achievements
- **Excessive:** May be overused or relied upon too heavily
- **Strong/Rough:** Operates like a hammer — direct and powerful
- **Self-confident:** But at the same time cautious
- **Narcissistic:** May have self-centered tendencies

## Example: IT Audience

A programmer with a dominant first function in Logic may:
- Excel at integrating information from their primary psychosophic aspect (Logic)
- Create a cohesive understanding of programming problems
- Struggle with breaking down complex problems into smaller components
- Have difficulty understanding nuances in each aspect

## Time Categories

The model relates personality to four temporal categories:

- **Present:** Action-oriented, here-and-now focus
- **Past:** Memory, tradition, experience-based
- **Future:** Vision, intuition, possibility-oriented
- **Eternity:** Reflection, philosophical, values-driven
