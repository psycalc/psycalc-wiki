# IEI — Intuitive Ethical Introvert (Yesenin)

**Socionics type:** IEI (INFp)  
**Quadra:** Beta  
**MBTI equivalent:** INFP  
**Function Stack:** 1Ni 2Fe 3Si 4Te 5Se 6Ti 7Ne 8Fi

## Function Profile

| Function | Element | Description |
|----------|---------|-------------|
| **1st (Base)** | Ni | Time, meaning, trajectory, symbolic pattern |
| **2nd (Creative)** | Fe | Emotional atmosphere, expression, dramatic modulation |
| **3rd (Role)** | Si | Comfort and bodily steadiness under social expectation |
| **4th (PoLR)** | Te | Efficiency pressure, procedural demands |
| **5th (Suggestive)** | Se | Decisive force, protection, momentum |
| **6th (Activation)** | Ti | Structure, principled explanation |
| **7th (Ignoring)** | Ne | Alternative possibilities |
| **8th (Demonstrative)** | Fi | Personal relation calibration in the background |
