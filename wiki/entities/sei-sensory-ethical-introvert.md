# SEI — Sensory Ethical Introvert (Dumas)

**Socionics type:** SEI (ISFp)  
**Quadra:** Alpha  
**MBTI equivalent:** ISFP  
**Function Stack:** 1Si 2Fe 3Ti 4Te 5Ne 6Fi 7Se 8Ni

## Function Profile

| Function | Element | Description |
|----------|---------|-------------|
| **1st (Base)** | Si | Comfort, body state, sensory atmosphere |
| **2nd (Creative)** | Fe | Emotional atmosphere, expression, social warmth |
| **3rd (Role)** | Ti | Formal structure, consistency |
| **4th (PoLR)** | Te | Efficiency pressure, productivity metrics |
| **5th (Suggestive)** | Ne | Possibilities, fresh perspectives |
| **6th (Activation)** | Fi | Personal warmth, relational confidence |
| **7th (Ignoring)** | Se | Force, pressure, direct control |
| **8th (Demonstrative)** | Ni | Timing, background forecasting |
