# Socionics Plus Minus Signs

## Overview

In some Socionics traditions, functions are assigned **plus (`+`)** and **minus (`-`)** signs. These signs are used to distinguish two variants of the same information element depending on which other elements they are blocked with in a type's structure.

This convention is not universally accepted across all schools of Socionics, but it is a recognized part of some source traditions.

## Core Rule

According to Wikisocion:

- **Alpha and Gamma**: irrational functions are given `+`, rational functions `-`
- **Beta and Delta**: irrational functions are given `-`, rational functions `+`

This is a rule about the sign convention within a type's structure, not a claim that one sign is stronger or better.

## Alternate Names

Some authors use alternate labels:

- `+` = short-range or expanding
- `-` = long-range or reducing

These labels are not always intuitive, so the plain `plus/minus` notation is safer.

## Practical Meaning

The plus/minus system suggests that the same information element can be expressed with a different directional emphasis depending on the rest of the type structure.

Wikisocion frames this as a convention for separating two "versions" of the same IM element.

## Caveat

- This is not a universally accepted Socionics standard.
- Some socionists consider the practice logically superfluous.
- Use signs as a **school-specific analytic layer**, not as a universal fact.

## Use In This Project

- The project may store function-sign information when a source-backed type-specific assignment is available.

## Type-Specific Derivation Example: SLI

For **SLI**, the current project uses the following source-backed derivation:

1. `SLI` belongs to **Delta** quadra.
2. In the Wikisocion convention, **Beta and Delta** give:
   - irrational functions → `-`
   - rational functions → `+`
3. SLI Model A stack in the current wiki is:
   - `1Si 2Te 3Fi 4Ne 5Ni 6Fe 7Ti 8Se`

Applying the convention yields:

- `1Si-`
- `2Te+`
- `3Fi+`
- `4Ne-`
- `5Ni-`
- `6Fe+`
- `7Ti+`
- `8Se-`

This assignment is valid **within the plus/minus school convention documented here**. It should not be presented as a universally accepted Socionics fact.
