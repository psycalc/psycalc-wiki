# Socionics Function Dichotomies

## Overview

This page summarizes the core dichotomies used to describe Model A functions in classical Socionics source traditions.

## Core Dichotomies

### Mental / Vital

- **Mental:** 1, 2, 3, 4
- **Vital:** 5, 6, 7, 8

Mental functions are more verbalized and consciously framed. Vital functions are more backgrounded, embodied, and less directly verbalized.

### Accepting / Producing

- **Accepting:** 1, 3, 5, 7
- **Producing:** 2, 4, 6, 8

Accepting functions register and take in a slice of reality first. Producing functions shape or respond based on what the paired accepting function has taken in.

### Strong / Weak

- **Strong:** 1, 2, 7, 8
- **Weak:** 3, 4, 5, 6

Strong functions are more differentiated and robust under pressure. Weak functions depend more on context and external support.

### Inert / Contact

- **Inert:** 1, 4, 6, 7
- **Contact:** 2, 3, 5, 8

Inert functions preserve their internal contour and are harder to reshape from outside. Contact functions adapt more situationally and interface more directly with the environment.

### Valued / Subdued

- **Valued:** 1, 2, 5, 6
- **Subdued / Unvalued:** 3, 4, 7, 8

Valued functions define quadra values and felt importance. Subdued functions are usable, sometimes very strong, but not preferred as the main mode of life.

### Evaluatory / Situational

- **Evaluatory:** 1, 4, 5, 8
- **Situational:** 2, 3, 6, 7

Evaluatory functions make firmer judgments. Situational functions act more case-by-case and depend more on context.

### Bold / Cautious

- **Bold:** 1, 3, 6, 8
- **Cautious:** 2, 4, 5, 7

Bold functions are used more freely. Cautious functions are more guarded and sensitive to criticism.

### Verbal / Nonverbal

- **Verbal:** 1, 2, 5, 6
- **Nonverbal:** 3, 4, 7, 8

Verbal functions are more discussable and socially exchanged. Nonverbal functions tend to show through behavior, reaction, and practical action.

### Exterior / Interior

- **Exterior:** 1, 3, 6, 8
- **Interior:** 2, 4, 5, 7

This dichotomy appears in some source traditions and should be treated as secondary compared to the core Model A block structure.

## Dimensionality

Wikisocion presents the following standard dimensional mapping:

- **4D:** 1, 8
- **3D:** 2, 7
- **2D:** 3, 6
- **1D:** 4, 5

Practical reading:

- **4D:** broad, confident mastery
- **3D:** strong situational competence
- **2D:** workable but limited under demand
- **1D:** narrowest and most dependent on external support

## Important Caveat on Function Signs
