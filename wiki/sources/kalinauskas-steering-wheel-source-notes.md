# Kalinauskas Steering Wheel Source Notes

## Overview

This page summarizes accessible online sources for the Socionics concept known as **штурвал Калинаускаса** / **Kalinauskas steering wheel**.

The source base is uneven: the most direct authorial-looking material is available through a third-party book mirror, while the most organized online exposition is a dedicated secondary site based on Grigory Reinin's lectures/book materials.

## Main Source Findings

### Dedicated site framing

The dedicated site states that it is focused on **the Kalinauskas steering-wheel model rather than Model A**. It names Reinin's lectures, published as *Соционика: Типология. Малые группы.* in 2005, as the main source.

This supports treating the steering wheel as a school-specific interpretive model rather than as the canonical Model A.

### Kalinauskas excerpt framing

The book mirror presents a section titled **“Штурвал Калинаускаса.”** It describes the steering wheel as a structure that helps perception, type identification, and understanding interconnections within Socionics.

The excerpt connects the model with the **method of qualitative structures** and uses the example of ILE / Don Quixote and ILI / Balzac to discuss opposite-type relations and problem/solution dynamics.

### Eight labels

The clearest online schema gives eight labels:

| Label | Name | Function in the steering-wheel schema |
|-------|------|----------------------------------------|
| I | Хорошо | principle of existence; level of organization |
| -I | Плохо | ignored zone; minus-values |
| II | Надо | creativity; functioning |
| -II | Не надо | norms and stereotypes |
| III | Проблемы | self-evaluation; least-resistance/problem zone |
| -III | Решения | solutions; self-realization/protection of self-esteem |
| IV | Хочу | motives; suggestive zone; evaluation of place |
| -IV | Боюсь | fears; suspiciousness/anxiety themes |

## Source-Status Caveats

- The model is not documented in this wiki as universal Socionics consensus.
- The strongest direct text is currently from a third-party book mirror.
- The dedicated site is useful but secondary and school-specific.
- Claims about “resonance,” suggestive influence, and shifting into other type languages should be treated as authorial practice language, not empirical psychology.
- The opposite-type emphasis diverges from the more common duality-centered Socionics framing.
