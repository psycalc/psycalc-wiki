# Agentic Research Pipelines: Полный обзор

## Что это?

**Agentic pipeline (воркфлоу)** = цепочка AI-агентов, где каждый делает свою задачу и передаёт результат следующему.

Аналогия: Конвейер на заводе
```
[Агент 1: Поиск] → [Агент 2: Анализ] → [Агент 3: Код] → [Агент 4: Отчёт]
```

**Что умеет:**
- Читает 50 статей за ночь
- Пишет и запускает Python код
- Генерирует LaTeX статью
- **AgentRxiv:** агенты делятся кодом друг с другом

**Стоимость:** ~$7-13 за исследование (gpt-4o)

**Как запустить:**
```bash
git clone https://github.com/SamuelSchmidgall/AgentLaboratory.git
cd AgentLaboratory
python ai_lab_repo.py --yaml-location "experiments/your_idea.yaml"
```

## Инструменты для построения pipeline

### Без кода (No-code)

| Инструмент | Что делает | Цена |
|-----------|-----------|------|
| **Make.com** | Визуальный конструктор | Бесплатно до 1000 запусков |
| **Zapier** | Автоматизация между сервисами | Бесплатно до 100 |
| **LangFlow** | Визуальный AI-конвейер | Бесплатно (open source) |

### С кодом

| Фреймворк | Язык | Сложность |
|-----------|------|-----------|
| **LangGraph** | Python | Средняя |
| **CrewAI** | Python | Низкая |
| **AutoGen** | Python | Средняя |
| **Agno** | Python | Низкая |
| **Microsoft Semantic Kernel** | C#/Python | Средняя |

### Пример на CrewAI (5 минут)

```python
from crewai import Agent, Task, Crew

# Агент для поиска
researcher = Agent(
    role="Researcher",
    goal="Find studies on socionics and relationship satisfaction",
    backstory="Expert in psychology research methods",
    tools=[search_tool, read_tool]
)

# Агент для анализа
analyst = Agent(
    role="Data Analyst", 
    goal="Analyze if dual pairs are happier",
    backstory="Expert in statistical analysis",
    tools=[analysis_tool]
)

# Агент для написания
writer = Agent(
    role="Scientific Writer",
    goal="Write research report",
    backstory="PhD in psychology, publishes in journals",
    tools=[write_tool]
)

# Создаём задачи
research_task = Task(
    description="Find all studies on socionics intertype relations",
    agent=researcher
)
analysis_task = Task(
    description="Test hypothesis: dual pairs are happier",
    agent=analyst,
    context=[research_task]
)
writing_task = Task(
    description="Write research report",
    agent=writer,
    context=[analysis_task]
)

# Запускаем
crew = Crew(agents=[researcher, analyst, writer], tasks=[research_task, analysis_task, writing_task])
result = crew.kickoff()
```

## Готовые исследования с pipeline

### The AI Scientist (Sakana AI)

**Pipeline:**
```
Идея → Код → Эксперимент → Анализ → Статья → Peer Review
```

**Результат:**
- Генерирует novel идеи
- Код запускается автоматически
- Статья генерируется в LaTeX
- Automated peer review проходит

**GitHub:** github.com/SakanaAI/AI-Scientist

### AgentRxiv (расширение Agent Laboratory)

Агенты делятся наработками:
```
Агент 1 (ночь 1) → загружает код на AgentRxiv
Агент 2 (день 2) → скачивает и дорабатывает
Агент 3 (день 3) → добавляет эксперименты
```

## Pipeline для типологий: план

### День 1: Обзор литературы

**Pipeline:**
```
ChatGPT (гипотеза) 
    ↓
GPT Researcher (поиск 20+ статей)
    ↓
Claude (резюме)
    ↓
Протокол исследования
```

### День 2: Эксперимент

**Pipeline:**
```
ChatGPT (анкета) 
    ↓
Google Forms (рассылка)
    ↓
Python Script (анализ)
    ↓
Statistical tests
```

### День 3: Статья

**Pipeline:**
```
Claude (черновик)
    ↓
Grammarly (проверка)
    ↓
LaTeX (оформление)
    ↓
Overleaf (публикация)
```
